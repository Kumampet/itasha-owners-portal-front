# プライバシー・利用規約

- **URL**: `/privacy`, `/term`
- **実装**: `src/app/privacy/page.tsx`, `src/app/term/page.tsx`
- **認証**: 不要

## 性質

法務・ポリシー文言の静的ページ。アプリ機能からのリンク先として維持する。公開トップ・サービス概要と同様に `SiteHeader` と `PublicSiteFooter` を表示する。本文はカード枠なしのフラットレイアウトとし、各章見出しはホームのイベントカラム見出しと同系の左グラデーションバー（`SectionHeadingWithAccent`）で装飾する。

## 変更時

文言変更はコンテンツレビュー前提。ルートパスを変えない（フッター・外部参照がある）。
